# 픽셀 도안 인쇄 — 어르신 인지학습 활동지

숫자 색칠(컬러 바이 넘버) 픽셀아트 도안을 **A4/A3 종이로 인쇄**하는 도구입니다.
서버·로그인·외부 라이브러리 없는 정적 웹페이지로, 어르신 인지학습용 활동지를
색연필 12색 기준으로 출력합니다.

## 사용법

1. 도안을 고른다 (난이도별로 정렬)
2. 용지(A4/A3)·방향(자동/세로/가로)·배경 칸·이름날짜 칸을 고른다
3. 미리보기를 확인하고 **🖨️ 인쇄하기** — 컬러 인쇄 권장 (범례 색 표시)

## 도안 구성 (43종)

- **그림 도안 14종** — 하트·별·무당벌레·토끼·튤립·무지개·아이스크림·로켓·나비·딸기·물고기·팬더·공룡·케이크
- **활동지 29종** (18×16, 모든 칸 색칠, 색연필 9~10색)
  - 원본 활동지 9장: 셈하기(2+8·86+4·66+4·56+8)·글자 놀이 2장·알록달록 하트·겨울·입춘
  - 신규 20장: 해님·사과·병아리·나비·튤립·우산·눈사람·수박·차·집·별·물고기·포도·복주머니·단풍잎·달팽이·행복·건강·사랑·숫자 123

## 인쇄 옵션

| 옵션 | 설명 |
|---|---|
| 색상 | **색연필 12색**(비슷한 색은 합치고 흰 칸은 비움, 색연필 이름 표시) / 원본 색 |
| 배경 칸 | **비우기** / **배경도 색칠**(빈 칸을 안 쓰인 색연필 번호로 채워 활동지 스타일로) |
| 이름·날짜 칸 | 학습지 상단 기입란 넣기/빼기 |

## 개발

```
index.html / print.css / print.js   # 인쇄 툴
pictures/*.js                       # 도안 (그림 1개 = 파일 1개)
tools/validate-picture.js           # 도안 계약 검증: node tools/validate-picture.js
```

도안 계약: `PIXELS.push({ id, name, emoji, category, level, palette: ['#hex', ...], rows: ['0120...', ...] })`
— rows의 문자 '0'~'9'가 palette 인덱스, 표시 번호는 인덱스+1.
