# 아기 그림일기 👶📔

그림일기 포맷으로 아기의 하루를 기록하는 모바일 우선 웹앱.
단일 HTML 정적 파일 — 별도 빌드/서버 없이 브라우저에서 바로 동작합니다.

## 구성

| 경로 | 설명 |
|------|------|
| `index.html` | **실제 앱** — 사진 레이아웃 엔진 + 그림일기 |
| `design/index.html` | **디자인 시안 갤러리** — 5종 무드 비교 |
| `design/sian1~5.html` | 개별 디자인 시안 |
| `PROGRESS.md` | 팀별 진행상황 / 검증 결과 |

## 그림일기 한 페이지 구성

- **헤더**: 날짜 · 날씨(☀️⛅☁️🌧️❄️) · 아기 디데이 뱃지
- **제목**
- **사진 영역**: 4~8장
  - 장수별 프리셋 레이아웃(격자/대표+N/지그재그 등)
  - **AI 자동배치**: 각 사진의 가로·세로 방향을 분석해 가장 어울리는 프리셋을 고르고, 대표(가로·큰) 사진을 큰 셀에 배정
  - **수동 배치**: 사진 셀을 길게 눌러 끌면 자리 교체
  - **사진 꾸미기**: 필터(뽀샤시/따뜻/시원/선명/세피아/흑백) · 밝기·채도 · 프레임(화이트/원형/그림자/테이프) · 이모지 스티커
- **본문**: 원고지 레이아웃
  - **타이핑** 입력
  - **펜슬** 입력(색상·굵기·지우개)

## 저장

모든 기록은 브라우저 `localStorage`에 자동 저장됩니다(4초 간격 + 저장 버튼).
`🖨️` 버튼으로 인쇄/PDF 저장이 가능합니다.

## 사용한 기술

순수 HTML/CSS/JavaScript. 외부 의존성 없음(웹폰트만 Google Fonts).
행 기반 flex 타일링으로 레이아웃이 항상 빈틈없이 채워지고, Pointer Events로 드래그 재배치와 펜 드로잉을 처리합니다.
