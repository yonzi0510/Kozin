# 육아 그림일기 앱 — 진행상황 (총괄팀)

> 아기의 하루를 그림일기 포맷으로 기록하는 웹앱. 모바일 우선, 단일 HTML 정적 배포(GitHub Pages).

## 📋 제품 요구사항

그림일기 한 페이지 구성:
- **상단 헤더**: 날짜 · 날씨 · 아기 D-day(디데이)
- **제목** 입력
- **사진 영역**: 4~8장 배치
  - 수동 배치 변경 가능
  - AI 자동 배치
  - 사진 꾸미기(필터/프레임/스티커)
- **본문**: 원고지 레이아웃
  - 타이핑 입력
  - 펜슬(손글씨) 입력

## 👥 팀별 업무 분담

| 팀 | 업무 | 상태 |
|----|------|------|
| 총괄팀 | 프로젝트 구조 · 진행 관리 · 통합 | ✅ 완료 |
| 디자인팀 | 디자인 시안 5종 제작 | ✅ 완료 (5/5) |
| 개발팀 | 사진 레이아웃 엔진 개발 | ✅ 완료 (브라우저 검증 통과) |

## 🎨 디자인 시안 (5종)

1. **파스텔 크레용** — 따뜻하고 아기자기한 크레용 감성
2. **미니멀 화이트** — 여백 중심의 깔끔한 미니멀
3. **크라프트 스크랩북** — 크라프트지 + 마스킹테이프 콜라주
4. **수채화 감성** — 부드러운 수채 번짐 배경
5. **뉴트로 원고지** — 빈티지 원고지 감성

시안 갤러리: `diary/design/index.html`

## 🧩 개발 산출물

- `diary/index.html` — 실제 동작 앱 (사진 레이아웃 엔진 포함)
  - 4~8장 프리셋 레이아웃 + 수동 드래그 재배치
  - AI 자동 배치 (사진 비율/장수 기반 최적 배치)
  - 사진 꾸미기 (필터 · 프레임 · 스티커)
  - 원고지 본문 (타이핑 + 펜 드로잉)

## ✅ 검증 결과 (개발팀 QA)

실제 브라우저(Chromium)에서 자동 스모크 테스트 통과:
- 사진 6장 주입 → 6칸 정상 렌더링
- AI 자동배치: 방향 분석 후 기본 격자 → `대표+5` 레이아웃으로 재배치, 가로 사진을 대표 셀에 배정
- 레이아웃 피커 3종 / 필터 7종 · 프레임 5종 · 스티커 16종 노출
- 펜 캔버스 정상 초기화, 타이핑↔펜 모드 전환
- 디데이 계산 정확 (`하윤 태어난지 D+128`)
- 콘솔 에러 없음 (구글폰트 로드만 샌드박스 네트워크 차단 → 실제 배포 시 정상)

## 🚀 실행 방법

- 디자인 시안 갤러리: `diary/design/index.html` 열기
- 실제 앱: `diary/index.html` 열기 (모바일에서 보기 권장)
- GitHub Pages 배포 시: `.../diary/` (앱), `.../diary/design/` (시안 갤러리)

## 📅 진행 로그

- 프로젝트 구조 생성, 요구사항 정리, 팀 착수
- 디자인팀: 시안 5종(파스텔·미니멀·크라프트·수채화·뉴트로) 병렬 제작 완료
- 개발팀: 사진 레이아웃 엔진 + 그림일기 앱(`index.html`) 개발 및 브라우저 검증 완료
- 총괄팀: 시안 갤러리(`design/index.html`) 조립, 통합, 문서화 완료
